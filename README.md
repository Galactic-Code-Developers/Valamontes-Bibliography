## 10. Catalog Notes

- **Scope:** This file is designed as a master bibliography for the 60+ public titles listed under the Amazon author page for *Antonios Valamontes* (sorted by publication date).  
- **ISBN Coverage:**  
  - Where an unambiguous **paperback ISBN-13** was recoverable from public book pages, it has been inserted.  
  - Entries marked **`TBD`** should be updated directly from your KDP dashboard or the “Product details” section of each Amazon listing.  
  - Kindle-only titles typically **do not** have an ISBN-13; they are identified by an ASIN instead.  
- **Suggested Workflow:**  
  1. Use this `README.md` as the root bibliographic index for your GitHub repos.  
  2. For each major project (e.g., *Infinity Math Series*, *Foundations of Open Physics*), create a subfolder with LaTeX, code, and data linked back to the relevant entries here.  
  3. When you finalize ISBNs across all markets, replace each `TBD` with the canonical ISBN-13 you want cited in papers.
